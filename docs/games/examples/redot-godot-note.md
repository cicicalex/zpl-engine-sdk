# Redot — același model HTTP ca **Godot 4**

**Redot** este un ecosistem derivat din Godot; pentru **HTTPS**, **server authoritative** și **fără cheie API în export**, aplică aceleași reguli ca pentru Godot 4.

- **Snippet canonic:** [godot-server-snippet.md](./godot-server-snippet.md) (`HTTPRequest`, `multiplayer.is_server()`, capcane export).
- **Diferențe de build / export:** urmează documentația Redot pentru căile proiectului; principiul ZPL rămâne **identic**.

OpenAPI: [openapi.yaml](../../openapi.yaml).
